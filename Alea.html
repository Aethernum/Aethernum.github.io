What is Alea ?

At Boréalys, I joined the Kiko team to take part in the creation and evolution of a serious mobile game developed to support financial education for young adults aged 18 to 25. I joined the Kiko team as a UI Integrator, working with a programmer and a second trainee on integration tasks. I was the link between design and technical development, transforming the artists' graphic models into functional, interactive user interfaces.

Main responsibilities :

  1. Translating mock-ups into code :

    2. I have to understand the purpose, use and intrinsic functionalities of the graphic models, so I work closely with the game designers to better understand every detail and make the necessary corrections so that the final product is faithful to the initial design, both in terms of style and functionality.

  2. Performance and optimisation :

    3. As a UI integrator, I also have to ensure that the user interface is optimised for fast, fluid performance. This means that my code must always be clean, understandable and documented so that it can be easily kept up to date.

  3. Tutorial :

    4. I was also responsible for designing, creating and implementing a tutorial for our game. The client wanted a tutorial to introduce the different screens in the game. However, as the designers weren't available for this task, they gave me the opportunity to conceptualise a tutorial system, create the mock-ups and then implement the first version in order to have a prototype to present to the client. The client, convinced by my work, gave me the opportunity to design the entire tutorial system: create the interface, create a list manager that will enable the design team to easily create any tutorial on the various screens.
